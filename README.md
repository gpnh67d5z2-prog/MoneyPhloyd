# MoneyPhloyd - Конвертер валют

Современное веб-приложение для конвертирования валют в реальном времени с красивым интерфейсом и поддержкой популярных мировых валют.

## Возможности

- Конвертирование между 7+ мировыми валютами
- Актуальные курсы обмена в реальном времени
- Современный тёмный интерфейс с красной цветовой схемой
- Мгновенные расчёты при изменении суммы
- Кнопка для быстрого обмена валют местами
- Адаптивный дизайн для всех устройств
- Поддержка русского языка

## Поддерживаемые валюты

- USD - Доллар США
- EUR - Евро
- GBP - Британский фунт
- JPY - Японская иена
- RUB - Российский рубль
- CNY - Китайский юань
- INR - Индийская рупия

## Быстрый старт

1. Откройте файл index.html в веб-браузере
2. Введите сумму для конвертирования
3. Выберите валютные пары
4. Посмотрите результат в реальном времени

## API

Приложение использует бесплатный API от **Exchange Rate API** для получения актуальных курсов обмена валют в реальном времени.

Источник: https://exchangerate-api.com/

Endpoint: \https://api.exchangerate-api.com/v4/latest/{currency}\

Описание: Каждый запрос получает текущие курсы для выбранной валюты. API возвращает объект с курсами конвертирования в другие валюты. Приложение использует эти данные для автоматического расчёта и отображения результатов конвертирования.
